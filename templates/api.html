Come Usare l'API per Fermata Bus

Introduzione

Questa API permette di ottenere informazioni in tempo reale per una fermata bus specifica. È possibile utilizzare il comando curl per inviare una richiesta POST con l'ID della fermata e ottenere i dettagli relativi ai passaggi dei bus.

Comando Curl

Per ottenere le informazioni sulla fermata con ID 597, puoi utilizzare il seguente comando curl: 

curl -X POST http://127.0.0.1:5000/fermata/597

Output dell'API

Quando invii la richiesta, l'API restituisce un output che contiene i dettagli della fermata, le linee di bus che passano e gli orari di passaggio. Un esempio di output è il seguente:

Output:
Fermata: BERNINI
Linea: 9 (BARRIERA LANZO, PIAZZA STAMPALIA)
Passaggi: 19:41* 19:48* 20:02* 20:16*
Prossimo passaggio: 10 minuti
Linea: 16CD (CIRCOLARE DESTRA, PORTA PALAZZO)
Passaggi: 19:47* 20:03* 20:23*
Prossimo passaggio: 16 minuti
